Script to generate Beckhoff PLC functions to determine the time of ...  
* sunrise
* sunset
* noon  
* dawn 
* dusk 

... based on longitude and longitude.


May also work with IEC 61131 sytems from other vendors. (Especially CODESYS based PLCs )
